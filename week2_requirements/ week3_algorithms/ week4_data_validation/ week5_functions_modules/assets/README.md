# Year 11 Software Engineering PRIMM + Starter Repo Learning Hub

## Purpose
This hub supports Weeks 2–5 of the Programming Fundamentals unit.
You will use PRIMM:
- Predict
- Run
- Investigate
- Modify
- Make

## Learning goals
By using this hub, you will:
- read and explain Python code
- test and modify code safely
- use GitHub workflows and organised files
- build towards your puzzle support tool project

## Weekly sections
- [Week 2: Requirements and specifications](week2_requirements/task.md)
- [Week 3: Algorithms and control structures](week3_algorithms/annotated_code.py)
- [Week 4: Data representation and validation](week4_data_validation/validation_exemplar.py)
- [Week 5: Functions, modules and starter repo](week5_functions_modules/starter_repo/README.md)

## How to use this hub
1. Open the week folder your teacher tells you to use.
2. Start with Predict.
3. Then complete Run, Investigate, Modify and Make.
4. Save your work in the correct file.
5. Upload screenshots or responses if instructed.

## Accessibility and support
- Use the downloadable PDF version if needed.
- Captions are provided on all screencasts.
- Ask for scaffolded or extension tasks if your teacher directs you to them.

## Evidence of learning
This hub supports formative checkpoints including:
- code reading
- tracing
- testing
- commit history
- README/help text updates
